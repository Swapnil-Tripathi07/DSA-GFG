# [Last index of One](https://www.geeksforgeeks.org/problems/last-index-of-15847/1?sortBy=submissions&category%5B%5D=Searching&page=1&difficulty%5B%5D=-1)
## Easy
Given a string s consisting of only '0's and '1's,&nbsp; find the last index of the '1' present.
Note:&nbsp;If '1' is not present, return "-1"
Examples:
Input: s = 00001
Output: 4
Explanation: Last index of&nbsp; 1 in given string is 4.

Input: s = 0
Output: -1
Explanation: Since, 1 is not present, so output is -1.
Expected Time Complexity: O(n)Expected Auxillary Space: O(1)
Constraints: 1 &lt;= |s| &lt;= 106s = {0,1}

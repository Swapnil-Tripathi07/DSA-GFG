# [Delete Alternate Nodes](https://www.geeksforgeeks.org/problems/delete-alternate-nodes/1?sortBy=submissions&category%5B%5D=Linked%2520List&page=1&difficulty%5B%5D=-1)
## Easy
Given a Singly Linked List, Delete all alternate nodes of the list ie delete all the nodes present in even positions.
Examples :
Input: LinkedList: 1-&gt;2-&gt;3-&gt;4-&gt;5-&gt;6 Output: 1-&gt;3-&gt;5
Explanation: Deleting alternate nodes ie 2, 4, 6 results in the linked list with elements 1-&gt;3-&gt;5.

Input: LinkedList: 99-&gt;59-&gt;42-&gt;20 Output: 99-&gt;42 
Expected Time Complexity: O(n)Expected Auxiliary Space:&nbsp;O(1)
Constraints:1 &lt;= number of nodes &lt;= 1051 &lt;= node-&gt;data &lt;= 103

# [Longest substring with distinct characters](https://www.geeksforgeeks.org/problems/longest-distinct-characters-in-string5848/1?sortBy=submissions&category%5B%5D=Strings&page=1&difficulty%5B%5D=0)
## Medium
You are given a string s.&nbsp;You have to find the length of the longest substring with all distinct characters.&nbsp;
Examples:
Input: s = "geeksforgeeks"
Output: 7
Explanation: "eksforg" is the longest substring with all distinct characters.

Input: s = "aaa"
Output: 1
Explanation: "a" is the longest substring with all distinct characters.
Input: s = "abcdefabcbb"
Output: 6
Explanation: The longest substring with all distinct characters is "abcdef", which has a length of 6.
Constraints:1 ≤ s.size() ≤ 105s consists of lowercase english letters.
